# Typography & Fonts

This directory contains the official kieks.me GbR corporate fonts and typography guidelines.

## Font Files

Corporate fonts should include:
- **Primary typeface** - Main font family for headers and body text
- **Secondary typeface** - Supporting fonts for specific use cases
- **Web fonts** - WOFF, WOFF2 formats for web use
- **Desktop fonts** - OTF, TTF formats for documents and design

## Font Weights & Styles

Include all necessary weights and styles:
- Regular
- Bold
- Italic
- Bold Italic
- Light (if applicable)
- Medium (if applicable)

## Typography Guidelines

- Font sizes for different heading levels
- Line height and letter spacing specifications
- Font pairing recommendations
- Accessibility considerations for readability

## Licensing

Ensure all fonts included have appropriate licenses for:
- Desktop use
- Web embedding
- Distribution within company materials
