# Logos

This directory contains official kieks.me GbR logos in various formats.

**Note**: Assets are also available as a complete ZIP archive. Download all logo assets as part of the [latest release ZIP](https://github.com/kieksme/kieks.me.cicd/releases).

## File Formats

- **SVG** - Scalable Vector Graphics (preferred for web and print)
- **PNG** - Portable Network Graphics (transparent backgrounds)
- **JPG** - JPEG format (solid backgrounds)

## Deprecated Files

The following files with spaces in their names are deprecated and should not be used:

- `kieks.me - dark - purple.svg` / `.png` (deprecated)
- `kieks.me - dark - red.svg` / `.png` (deprecated)
- `kieks.me - light - red.svg` / `.png` (deprecated)
- `kieks.me - favicon.svg` / `.png` / `.jpg` (deprecated)
- `kieks.me - favicon light.svg` (deprecated)

Please use the current logo files listed above instead.

## Logo Variants

Logos are organized by:

- **Horizontal logos** - Full logo with wordmark (green, purple, red variants)
- **Single/Icon logos** - Icon-only versions (background, circle, transparent variants)
- **Dark/Light variants** - Optimized for different background colors

## Usage Guidelines

- Always maintain proper spacing around the logo (minimum clear space)
- Do not distort, rotate, or modify the logo
- Use the appropriate version for your background color
- Prefer SVG format for scalable graphics
- Use PNG for transparent backgrounds when needed
- Refer to the [brand guidelines](../guidelines/LOGO_USAGE.md) for detailed usage instructions
