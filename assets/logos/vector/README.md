# Vector Logos

This directory contains vector format logo files.

## Formats

- **SVG** (Scalable Vector Graphics): Best for web and digital use
- **AI** (Adobe Illustrator): Source files for editing
- **EPS** (Encapsulated PostScript): Universal format for professional design

## Benefits of Vector Formats

- Infinitely scalable without quality loss
- Small file sizes
- Easy to edit and modify
- Perfect for responsive web design
- Professional print quality at any size

## File Naming Convention

Use descriptive names that indicate the logo variation:
- `kieksme-logo-primary.svg`
- `kieksme-logo-white.svg`
- `kieksme-logo-black.svg`
- `kieksme-logo-monochrome.svg`
- `kieksme-icon.svg`
- `kieksme-wordmark.svg`

## Usage

Vector formats are preferred for:
- Web development
- Digital design
- Print production
- Signage and large format
- Any application requiring scalability

See [Logo Usage Guidelines](../../../guidelines/LOGO_USAGE.md) for detailed usage instructions.
