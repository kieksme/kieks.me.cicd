# Raster Logos

This directory contains raster format logo files.

## Formats

- **PNG**: With transparent background (preferred for most uses)
- **JPG**: With solid background (when transparency not needed)

## Resolution Guidelines

Provide multiple resolutions for different use cases:
- **@1x**: Standard resolution (e.g., 200px width)
- **@2x**: Retina/High-DPI displays (e.g., 400px width)
- **@3x**: Ultra-high resolution (e.g., 600px width)
- **Print**: 300 DPI minimum

## File Naming Convention

Include resolution in the filename:
- `kieksme-logo-primary@1x.png` (200px)
- `kieksme-logo-primary@2x.png` (400px)
- `kieksme-logo-primary@3x.png` (600px)
- `kieksme-logo-primary-print.png` (300 DPI)

## When to Use Raster

Raster formats are suitable for:
- Quick mockups and presentations
- Social media profile images
- Email signatures (PNG)
- Document insertion (Word, Google Docs)
- Specific size requirements

## Important Notes

⚠️ **Avoid scaling raster logos**:
- Enlarging causes pixelation and quality loss
- Always use the appropriate size or switch to vector

⚠️ **Use PNG for transparency**:
- PNG supports transparent backgrounds
- JPG always has a solid background

⚠️ **For print, use high resolution**:
- Minimum 300 DPI for professional printing
- Test print quality before final production

## Preferred Alternative

When possible, use vector formats (SVG, AI, EPS) from the `vector/` directory for better quality and flexibility.
