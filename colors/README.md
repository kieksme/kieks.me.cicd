# Brand Colors

This directory contains the official kieks.me GbR brand color palette.

## Color Specifications

Brand colors should be documented with:
- **Hex values** - For digital/web use
- **RGB values** - For digital displays
- **CMYK values** - For print materials
- **Pantone codes** - For professional printing

## Color Usage

- Primary colors - Main brand colors for major elements
- Secondary colors - Supporting colors for accents and highlights
- Neutral colors - Grays, blacks, and whites for text and backgrounds
- Accessibility - Ensure color combinations meet WCAG contrast requirements

## Files

This directory should contain:
- Color palette swatches (PNG, SVG)
- ASE (Adobe Swatch Exchange) files
- Color specification documents
