Font Pairing

Use Hanken Grotesk for headings and Source Sans 3 for body text to maintain brand consistency.

Heading Example

This is how the fonts work together. The heading uses Hanken Grotesk in bold, while the body text uses Source Sans 3 for optimal readability. This combination creates a clear visual hierarchy and maintains brand consistency across all communications.

Subheading Example

Smaller headings use medium weight, while body text remains consistent. This pairing ensures readability and visual appeal across all screen sizes and contexts.

Usage Examples

CSS Variables

body {
  font-family: var(--font-body);
  font-weight: 400;
}

h1, h2, h3, h4, h5, h6 {
  font-family: var(--font-heading);
}
