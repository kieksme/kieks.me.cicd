Company Pages • Career Pages • Brand Colors

LinkedIn Image Specifications

Alle Bilder für LinkedIn-Unternehmensseiten und Karriereseiten müssen den offiziellen LinkedIn-Spezifikationen entsprechen. Alle Bilder müssen PNG- oder JPEG-Dateien sein und dürfen maximal 3 MB groß sein.

Tab Modul Mindestgröße Empfohlene Größe
Seite Logobild 268 × 268 Pixel 400 × 400 Pixel
Seite Titelbild 4200 × 700 Pixel 4200 × 700 Pixel
Unternehmenskultur Hauptbild 1128 × 376 Pixel 1128 × 376 Pixel
Unternehmenskultur Eigene Module 502 × 282 Pixel 502 × 282 Pixel
Unternehmenskultur Firmenfotos 264 × 176 Pixel 900 × 600 Pixel
Beiträge Eigene Bilder 200 × 105 Pixel 1200 × 627 Pixel (1,91:1)

Wichtige Hinweise

  • Die Module des Tabs Unternehmenskultur sind nur verfügbar, wenn Sie Ihre Unternehmensseiten um Karriereseiten erweitert haben.
  • Die dynamische Präsentation des Titelbilds ist bei einer Mitgliedschaft für eine Premium-Unternehmensseite verfügbar.
  • Mitglieder können LinkedIn-Seiten auf unterschiedlichen Geräten und Bildschirmgrößen ansehen - beachten Sie die Safe Areas.
  • Wichtige Details sollten nicht zu nah am Rand des Titelbildes platziert werden, insbesondere nicht in der unteren rechten Ecke.
  • Ihr Logo-Bild wird sowohl auf hellem als auch auf dunklem Hintergrund angezeigt. Testen Sie Ihr Logo entsprechend.

Design Guidelines

Bildqualität

  • Verwenden Sie die empfohlenen Abmessungen für die beste Bildauflösung
  • Wählen Sie ein hochauflösendes JPEG anstelle einer PNG-Datei für große Bilder
  • PNG für Logos (Transparenz-Unterstützung)
  • Maximale Dateigröße: 3 MB

Safe Areas

  • Wichtige Inhalte (Logo, Text) in der Mitte platzieren
  • Titelbilder können auf verschiedenen Geräten zugeschnitten werden
  • Vermeiden Sie wichtige Inhalte in den Ecken, besonders unten rechts
  • Zentrierte Platzierung sorgt für Sichtbarkeit auf allen Bildschirmgrößen

Firmenfarben

LinkedIn Image Generator

Verwende das LinkedIn Image Generator Script, um LinkedIn-konforme Bilder automatisch zu erstellen. Das Script generiert Bilder mit Firmenfarben, Logo und optionalem Text.

Installation & Voraussetzungen

Das Script verwendet sharp für die Bildverarbeitung. Alle Dependencies sind bereits im Projekt installiert.

Verwendung

Das LinkedIn Image Generator Script kann auf zwei Arten verwendet werden:

  • Interaktiver Modus: Startet ohne Argumente - führt durch alle Schritte
  • CLI-Modus: Alle Parameter werden als Kommandozeilen-Argumente übergeben

Interaktiver Modus (empfohlen) 

# Startet den interaktiven Modus
pnpm run generate:linkedin

# Der interaktive Modus führt Sie durch:
# - Auswahl des Bildtyps (Logo, Titelbild, etc.)
# - Auswahl der Firmenfarbe
# - Option für Logo-Integration
# - Option für Text-Overlay
# - Auswahl des Ausgabepfads

CLI-Modus 

# Titelbild mit Aqua-Hintergrund und Text
pnpm run generate:linkedin \
  --type title \
  --color aqua \
  --text "kieks.me GbR" \
  --output output/linkedin-title-aqua.jpg

# Logo-Bild mit Navy-Hintergrund
pnpm run generate:linkedin \
  --type logo \
  --color navy \
  --output output/linkedin-logo-navy.png

# Post-Bild mit Fuchsia-Hintergrund
pnpm run generate:linkedin \
  --type post \
  --color fuchsia \
  --text "We're hiring!" \
  --output output/linkedin-post-fuchsia.jpg

# Interaktiver Modus (empfohlen)
pnpm run generate:linkedin

Parameter

  • --type <type>: Bildtyp: logo, title, culture-main, culture-module, photo, post [erforderlich]
  • --color <color>: Firmenfarbe: aqua, navy oder fuchsia [erforderlich]
  • --logo <path>: Pfad zum Logo (optional, Standard: kieks.me Logo)
  • --text <text>: Text zum Anzeigen (optional)
  • --output <path>: Ausgabedatei-Pfad [erforderlich im CLI-Modus]
  • --format <format>: Ausgabeformat: jpeg oder png (Standard: jpeg für große Bilder, png für Logos)
  • --min: Verwendet Mindestgrößen statt empfohlener Größen

Bildtypen

  • logo: Logo-Bild: 400×400px (empfohlen) / 268×268px (min)
  • title: Titelbild: 4200×700px
  • culture-main: Unternehmenskultur Hauptbild: 1128×376px
  • culture-module: Unternehmenskultur Modul: 502×282px
  • photo: Firmenfoto: 900×600px (empfohlen) / 264×176px (min)
  • post: Post-Bild: 1200×627px (1,91:1 Seitenverhältnis)

Hilfe

Zeige die Hilfe mit: 

pnpm run generate:linkedin --help

Verwendungshinweise

Farbauswahl

  • Aqua: Für helle, freundliche Bilder
  • Navy: Für professionelle, seriöse Bilder
  • Fuchsia: Für kreative, auffällige Bilder
  • Wähle die Farbe basierend auf Kontext und Verwendungszweck

Dateigröße

  • Maximale Dateigröße: 3 MB (LinkedIn-Limit)
  • Das Script komprimiert automatisch, wenn nötig
  • JPEG für große Bilder (bessere Kompression)
  • PNG für Logos (Transparenz-Unterstützung)

Best Practices

  • Verwenden Sie die empfohlenen Größen für die beste Qualität
  • Platzieren Sie wichtige Inhalte in der Mitte (Safe Area)
  • Vermeiden Sie Text in den Ecken, besonders unten rechts
  • Testen Sie Logos auf hellem und dunklem Hintergrund
  • Verwenden Sie hochauflösende Bilder für beste Ergebnisse

Verwendungszwecke

  • LinkedIn-Unternehmensseiten
  • LinkedIn-Karriereseiten
  • LinkedIn-Beiträge und Posts
  • Unternehmenskultur-Module
  • Firmenfotos und Galerien

pnpm generate:image:linkedin:samples

Das Script generiert Beispiel-Bilder für alle LinkedIn-Bildtypen in allen drei Firmenfarben. Die Bilder werden im Verzeichnis examples/linkedin/ gespeichert.
